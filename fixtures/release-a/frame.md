# Release A · Frame system

## Overview

A warm editorial proof desk crossed with a precise motion workstation. The cream paper field keeps evidence human and legible; graphite makes the editor feel serious; cobalt appears only on active identity, verified action, and the story’s key words.

## Composition rules

- Use large asymmetric headlines and edge-anchored metadata.
- Keep visible source hashes, stable identities, times, and status language in IBM Plex Mono.
- Use cobalt as a functional signal, not ambient decoration.
- Preserve crisp borders and a single soft cobalt underprint; avoid glass, purple gradients, and glow-heavy “AI” styling.
- Every scene has a background, midground proof, and foreground metadata.

## Motion rules

- One focal event per beat.
- Entrances settle into a readable hold.
- Component state changes preserve one DOM subject.
- Timeline movement is linear; interface arrivals use controlled ease-out; overshoot is small and reserved for tactile selection.
